propósito del sitio

El propósito de este sitio es poderle permitir a cualquier usuario que ingrese, es darle a conocer que son y para qué sirven las Hojas de Estilo en Cascada CSS y los Frameworks CSS, para que en un futuro estos usuarios puedan crear sus propios sitios web, puesto que en este sitio encontrara una serie de recursos (AUDIOS y PDF), que le servirán de apoyo para adquirir conocimiento sobre los temas mencionados en esta web, y en un futuro este pueda lograr crear su propia web utilizando las hojas de estilo cascada y los frameworks.

Introduccion

Los CSS (Cascade Style Sheet) que traducido al epañol significa hojas de estilo en cascada css,es un estilo que nos permite controlar la apariencia de una página web. En el comienzo, los sitos web solo se concentraban en el contenido que en la presentación. Con los CSS podemos implementar estilos como por ejemplo: el tamaño, fuentes, color, espaciado entre los textos y recuadros así como el lugar donde colocar los texto e imagenes de nuestro sitio web. otro tema que hablaremos es de los frameworks css el cual es una estructura ordenada o guía de cómo se debe trabajar con un lenguaje, para los framework css estos tienen una estructura específica y herramientas para el diseño de una página web, y sirven para agilizar el desarrollo de páginas web que están basadas en estándares y compatibilidad en los navegadores y estos tienen códigos mas entendible.
